# Netflix Content Data Analysis

##  Overview

Analyzed streaming platform content data to understand content trends, identify audience preferences, and generate actionable business insights for improving content strategy and global growth.


##  Dataset

The dataset includes content attributes such as:

* Type (Movie / TV Show)
* Title
* Director
* Cast
* Country
* Release Year
* Rating
* Duration
* Genre (Listed In)
* Description


##  Objective

* Understand what type of content is available
* Identify popular genres and trends
* Analyze content distribution across countries
* Study audience targeting through ratings
* Provide data-driven recommendations for business growth


##  Key Analysis

* Exploratory Data Analysis (EDA)
* Univariate & Bivariate Analysis
* Distribution Analysis (Release Year, Duration)
* Category Analysis (Type, Genre, Rating, Country)
* Outlier Detection using Boxplots
* Correlation Analysis


##  Key Insights

* Movies dominate the platform compared to TV shows.
* Most content has been released after 2010, showing focus on recent content.
* Drama, International, and Comedy are the most common genres.
* Majority content is rated TV-MA and TV-14, targeting mature audiences.
* United States and India contribute the highest number of titles.


##  Content Trends

* Recent Content Focus: Majority of titles are modern (post-2010)
* Genre Concentration: Drama and International content dominate
* Audience Targeting: Strong focus on adult viewers
* Global Distribution: Content is concentrated in a few major countries


##  Tools Used

* Python (Pandas, NumPy)
* Matplotlib, Seaborn
* Google Colab


##  Business Impact

* Helps in deciding what type of content to produce
* Supports global expansion strategies through country analysis
* Identifies gaps in family-friendly content
* Improves content planning and audience targeting

* Performed end-to-end Exploratory Data Analysis on real-world dataset

* Used visual analysis to uncover trends and patterns
